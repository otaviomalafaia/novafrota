Privacidade e GDPR

Tratamos dados pessoais com transparência, minimização e segurança. Esta página descreve como recolhemos, utilizamos e protegemos as informações relacionadas com a NovaFrota.

Última atualização: Abril de 2025. Para exercer qualquer direito RGPD, contacte support@novafrota.pt.

Responsável

Controlador de dados

A NovaFrota é responsável pelo tratamento dos dados recolhidos neste site. O nosso Encarregado de Proteção de Dados (DPO) pode ser contactado através de support@novafrota.pt.

Base legal: consentimento explícito (Art.º 6.º, n.º 1, alínea a) do RGPD) e interesse legítimo para manter a segurança do serviço.

Minimização

Dados recolhidos

  • Email fornecido voluntariamente para envio de convites, novidades e respostas a pedidos de contacto.
  • Registo do momento do consentimento (timestamp) e user agent para auditoria de conformidade.
  • Hash do endereço IP (para auditoria de segurança) quando submete o formulário.
  • Metadados técnicos mínimos agregados e anonimizados para garantir a estabilidade do site.

Nunca partilhamos os seus dados com terceiros para marketing sem novo consentimento.

Direitos

Os seus direitos

  • Acesso, retificação ou apagamento dos dados pessoais.
  • Limitação ou oposição ao tratamento dos dados.
  • Portabilidade dos dados para outro responsável.
  • Retirada do consentimento a qualquer altura sem afetar a licitude do tratamento já efetuado.
  • Apresentar reclamação junto da autoridade de controlo (CNPD em Portugal).
Retenção

Retenção e cookies

Conservamos o seu email apenas enquanto mantivermos comunicações relevantes ou até exercer o direito de apagamento. Se nada ouvir da nossa parte em 24 meses, removemos o contacto.

Este site não utiliza cookies de rastreamento ou perfis comportamentais; apenas recursos técnicos essenciais. Os registos locais de consentimento guardados no seu navegador podem ser eliminados limpando a cache.

Processos

Como tratamos pedidos RGPD

  • Recolha: O formulário guarda email, consentimento, timestamp, user agent e hash do IP.
  • Exportação: Pode solicitar um resumo dos seus dados enviando email para support@novafrota.pt. Geramos o relatório a partir do endpoint protegido `/api/leads`.
  • Apagamento: Quando pedir para remover os seus dados, eliminamos o registo via endpoint seguro e confirmamos a conclusão por email.
  • Falhas de rede: Se o envio do formulário falhar, os dados ficam armazenados localmente no seu navegador (chave `novafrota-consents`). Envie-nos um email para concluir o processo manualmente.
